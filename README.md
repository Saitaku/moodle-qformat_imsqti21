moodle-qformat_imsqti21
========================

IMS QTI 2.1 Moodle import/export format
This plugin allow import and export of questions using the IMS QTI 2.1 standard.

Written by Laurent Opprecht from Geneva University
Retreived from https://code.google.com/p/moodle-qti-cp-fedora/
Upgraded for Moodle 2.9, 3.0 by Jean-Michel Védrine (vedrine@vedrine.net)

Note: Currently I have only worked on export and done very limited tests. In fact very tiny modifications were needed to adapt the code to Moodle recent changes.
